1  Übersicht

Diese Applikation berechnet die Kettenliniengeometrie von SBB-Oberleitungen für drei Systemtypen:

  • N-FL — fixabgefangen: Tragseil + Fahrdraht + Hänger, temperaturabhängige Ts-Zugkraft (Zustandsgleichung)
  • R-FL — beweglich abgefangen (Nachspannvorrichtung): konstante Zugkraft, keine Temperaturkorrektur nötig
  • Einzelleiter — einfache Kettenlinienberechnung eines einzelnen Leiters (Speiseleiter, Rückleiter o. ä.) ohne Fahrdraht und Hänger

Das Berechnungsmodell entspricht dem SBB-Referenzmodell Durchhang 2026.xlsm (VBA Upro1 / DurchhangNFL) und verwendet für bekannte FL-Kombinationen die VEM Handbuch Formeln 7.82 + 7.66 für den Fahrdraht-Durchhang (effektive Spannweite LT1 bei Höhendifferenz).

Cette application calcule la géométrie caténaire des lignes de contact SBB pour trois types de systèmes :

  • N-FL — ancrage fixe : câble porteur + fil de contact + pendotes, force de traction du CP dépendante de la température (équation d'état)
  • R-FL — ancrage mobile (tendeur automatique) : force de traction constante, aucune correction thermique nécessaire
  • Conducteur seul — calcul caténaire simple d'un conducteur unique (câble d'alimentation, câble de retour, etc.) sans fil de contact ni pendotes

Le modèle de calcul correspond au modèle de référence SBB Durchhang 2026.xlsm (VBA Upro1 / DurchhangNFL) et utilise pour les combinaisons FL connues la formule VEM Manuel 7.66 pour la flèche du fil de contact.

Questa applicazione calcola la geometria della catenaria delle linee di contatto FFS per tre tipi di sistema:

  • N-FL — ancoraggio fisso: fune portante + filo di contatto + pendoli, forza di tiro della FP dipendente dalla temperatura (equazione di stato)
  • R-FL — ancoraggio mobile (tendifilo automatico): forza di tiro costante, nessuna correzione termica necessaria
  • Conduttore singolo — calcolo semplice della catenaria di un singolo conduttore (cavo di alimentazione, cavo di ritorno, ecc.) senza filo di contatto né pendoli

Il modello di calcolo corrisponde al modello di riferimento FFS Durchhang 2026.xlsm (VBA Upro1 / DurchhangNFL) e utilizza per le combinazioni FL note la formula VEM Manuale 7.66 per la freccia del filo di contatto.

3  Berechnungsverfahren

Schritt 1  —  Verteilte Gesamtlast

qges = qTs + qFd + qHänger  [N/m]

Hänger: 0.020 kg/m × 9.81 m/s² = 0.196 N/m. Bei bekannter FL-Kombination: qges = MFL (exakterer VBA-Wert inkl. Klemmen).

Schritt 2  —  Stufe 1 Zustandsgleichung: unbelastet → belastet (N-FL)

Beim Anhängen von Fahrdraht und Hängern dehnt sich das Ts elastisch. Gleichgewicht via kubischem Polynom:

q²ges·Lm² / (24·H²b) − Hb/(E·A)Ts = q²Ts·Lm² / (24·H²ub) − Hub/(E·A)Ts

Lösung mit Newton-Raphson (nur E·A des Tragseils; VBA-konform). Ergebnis: HTs,belastet = F₀ (Referenzzustand).

Schritt 3  —  Kettenlinie Ausgangszustand (T₁)

Das Ts wird als exakte Kettenlinie modelliert mit Kettenlinienparameter a = H/q:

y(x) = a · cosh((x − x₀) / a) + C₀

Tiefpunkt x₀ und Konstante C₀ aus Randbedingungen h₁, h₂ bestimmt (exakte Lösung, keine Parabelannäherung).

Schritt 4  —  Stufe 2 Zustandsgleichung: Temperaturänderung (N-FL, VBA-konform)

Für bekannte FL-Kombinationen nach VEM Handbuch / VBA Upro1:

((MFL + ZL + MFL·HFd/F₀) / (HFd + x))² − (MFL/F₀)² − G·(α·ΔT + (x−F₀)/(E·A)) = 0

F₀ = HTs bei T₁ (belastet). G = 24/Lm² + (MFL/F₀)². ZL = Eislast [N/m] (0 wenn nicht anwendbar). Newton-Raphson → HTs,2.

Schritt 5  —  Kombinierte Kettenlinie & Durchhang (VEM Handbuch Formeln 7.82 + 7.66)

Massgebender Durchhang für Kontrollmessungen — kombiniertes Ts+Fd-System:

qvem = MFL·(1 + HFd/F₀) + ZL   [F₀ = HTs bei T₁, belastet]

LT1 = L + 2·|Δh|·(HTs,2+HFd) / (L·qvem)   [Formel 7.82, |Δh| = |h₂−h₁| ≥ 0]

fFd = LT1² / 8 · qvem / (HTs,2 + HFd)   [Formel 7.66 mit LT1]

Entspricht Zellenformeln P12/P14 in SBB Durchhang 2026.xlsm (NFL-Re Sheet). LT1 ≥ L immer (Absolutwert |Δh| — symmetrisch für h₁>h₂ und h₂>h₁). Gilt für T₁ und T₂.

Darstellung im Diagramm: Die angezeigte Ts-Kurve bei T₂ entspricht der kombinierten Kettenlinie des Ts+Fd-Systems mit den Parametern qvem und HTs,2+HFd. Diese stimmt mit der VEM-Parabel überein (Abweichung < 5 mm für übliche Spannweiten). Messpunkte (M1–M4) zeigen die Höhe auf dieser kombinierten Kurve.

Étape 1  —  Charge totale distribuée

qtot = qCP + qFC + qpendote  [N/m]

Pendote: 0.020 kg/m × 9.81 m/s² = 0.196 N/m. Pour combinaison FL connue: qtot = MFL (valeur VBA exacte incl. pinces).

Étape 2  —  Équation d'état niveau 1: non chargé → chargé (N-FL)

Lors du montage du fil de contact et des pendotes, le CP se déforme élastiquement. Équilibre via polynôme cubique:

q²tot·Lm² / (24·H²ch) − Hch/(E·A)CP = q²CP·Lm² / (24·H²nch) − Hnch/(E·A)CP

Résolution Newton-Raphson (E·A du câble porteur uniquement; conforme VBA). Résultat: HCP,chargé = F₀ (état de référence).

Étape 3  —  Caténaire état initial (T₁)

Le CP est modélisé comme caténaire exacte avec paramètre caténaire a = H/q:

y(x) = a · cosh((x − x₀) / a) + C₀

Point le plus bas x₀ et constante C₀ déterminés à partir des conditions aux limites h₁, h₂ (solution exacte, pas d'approximation parabolique).

Étape 4  —  Équation d'état niveau 2: variation de température (N-FL, conforme VBA)

Pour combinaisons FL connues selon VEM Manuel / VBA Upro1:

((MFL + ZL + MFL·HFC/F₀) / (HFC + x))² − (MFL/F₀)² − G·(α·ΔT + (x−F₀)/(E·A)) = 0

F₀ = HCP à T₁ (chargé). G = 24/Lm² + (MFL/F₀)². ZL = charge de glace [N/m] (0 si non applicable). Newton-Raphson → HCP,2.

Étape 5  —  Caténaire combinée & flèche (VEM Manuel formules 7.82 + 7.66)

Flèche déterminante pour les mesures de contrôle — système combiné CP+FC:

qvem = MFL·(1 + HFC/F₀) + ZL   [F₀ = HCP à T₁, chargé]

LT1 = L + 2·|Δh|·(HCP,2+HFC) / (L·qvem)   [Formule 7.82, |Δh|=|h₂−h₁| ≥ 0]

fFC = LT1² / 8 · qvem / (HCP,2 + HFC)   [Formule 7.66]

Correspond aux formules P12/P14 dans SBB Durchhang 2026.xlsm (feuille NFL-Re). LT1 ≥ L toujours (valeur absolue |Δh| — symétrique). Valable pour T₁ et T₂.

Affichage dans le diagramme: La courbe CP affichée à T₂ correspond à la caténaire combinée du système CP+FC avec les paramètres qvem et HCP,2+HFC. Les points de mesure (M1–M4) indiquent la hauteur sur cette courbe combinée.

Passo 1  —  Carico totale distribuito

qtot = qFP + qFC + qpendolo  [N/m]

Pendolo: 0.020 kg/m × 9.81 m/s² = 0.196 N/m. Per combinazione FL nota: qtot = MFL (valore VBA esatto incl. morsetti).

Passo 2  —  Equazione di stato livello 1: non caricato → caricato (N-FL)

Durante il montaggio del filo di contatto e dei pendoli, la FP si deforma elasticamente. Equilibrio tramite polinomio cubico:

q²tot·Lm² / (24·H²c) − Hc/(E·A)FP = q²FP·Lm² / (24·H²nc) − Hnc/(E·A)FP

Soluzione Newton-Raphson (solo E·A della fune portante; conforme VBA). Risultato: HFP,caricato = F₀ (stato di riferimento).

Passo 3  —  Catenaria stato iniziale (T₁)

La FP è modellata come catenaria esatta con parametro catenaria a = H/q:

y(x) = a · cosh((x − x₀) / a) + C₀

Punto più basso x₀ e costante C₀ determinati dalle condizioni al contorno h₁, h₂ (soluzione esatta, nessuna approssimazione parabolica).

Passo 4  —  Equazione di stato livello 2: variazione di temperatura (N-FL, conforme VBA)

Per combinazioni FL note secondo VEM Manuale / VBA Upro1:

((MFL + ZL + MFL·HFC/F₀) / (HFC + x))² − (MFL/F₀)² − G·(α·ΔT + (x−F₀)/(E·A)) = 0

F₀ = HFP a T₁ (caricato). G = 24/Lm² + (MFL/F₀)². ZL = carico di ghiaccio [N/m] (0 se non applicabile). Newton-Raphson → HFP,2.

Passo 5  —  Catenaria combinata & freccia (VEM Manuale formule 7.82 + 7.66)

Freccia determinante per le misure di controllo — sistema combinato FP+FC:

qvem = MFL·(1 + HFC/F₀) + ZL   [F₀ = HFP a T₁, caricato]

LT1 = L + 2·|Δh|·(HFP,2+HFC) / (L·qvem)   [Formula 7.82, |Δh|=|h₂−h₁| ≥ 0]

fFC = LT1² / 8 · qvem / (HFP,2 + HFC)   [Formula 7.66]

Corrisponde alle formule P12/P14 in SBB Durchhang 2026.xlsm (foglio NFL-Re). LT1 ≥ L sempre (valore assoluto |Δh| — simmetrico). Valido per T₁ e T₂.

Visualizzazione nel diagramma: La curva FP visualizzata a T₂ corrisponde alla catenaria combinata del sistema FP+FC con i parametri qvem e HFP,2+HFC. I punti di misura (M1–M4) indicano l'altezza su questa curva combinata.

5  Vordefinierte & benutzerdefinierte Kettenwerke

Die App enthält vordefinierte Kettenwerke für N-FL und R-FL. Daneben können über die ＋-Schaltfläche eigene Kettenwerke erfasst und gespeichert werden (lokaler Browser-Speicher). Benutzerdefinierte Kettenwerke erlauben die freie Eingabe von MFL, E, α, A sowie sh, Zugkräften und Abnutzungsgrenzwert.

L'application contient des caténaires prédéfinies pour N-FL et R-FL. En plus, il est possible d'enregistrer ses propres caténaires via le bouton ＋ (mémoire locale du navigateur). Les caténaires personnalisées permettent la saisie libre de MFL, E, α, A ainsi que sh, forces de traction et valeur limite d'usure.

L'applicazione contiene catenarie predefinite per N-FL e R-FL. Inoltre, è possibile registrare le proprie catenarie tramite il pulsante ＋ (memoria locale del browser). Le catenarie personalizzate permettono l'inserimento libero di MFL, E, α, A nonché sh, forze di tiro e valore limite di usura.

6  Diagnosewerkzeuge & Sonderfunktionen

6.1  Messpunkte M1 – M4

Im Bereich Messpunkte können bis zu vier beliebige x-Positionen (Abstände vom linken Mast) eingegeben werden. Die App berechnet für jeden Punkt:

  • Ts-Höhe über SOK (und in m ü. M., wenn SOK gesetzt)
  • Fahrdraht-Höhe über SOK
  • Ts-Durchhang an diesem Punkt

Im Diagramm erscheinen die Messpunkte als farbige Markierungen mit einem Callout-Label. Wenn SOK m ü. M. gesetzt ist, zeigt das Callout die absolute Höhe in m ü. M. als Hauptwert.

6.2  3-Punkte-Messung

Die 3-Punkte-Messung erlaubt die Rückrechnung der Ts-Zugkraft HTs aus einer Feldmessung. Eingaben:

  • xC — Messposition (Abstand vom linken Mast, Feldmitte empfohlen)
  • yC — gemessene Ts-Höhe an diesem Punkt, wahlweise in «m über SOK» oder in «m ü. M.» (wenn SOK gesetzt)

Die App löst numerisch nach HTs (Newton-Raphson auf der Kettenlinienformel), sodass die berechnete Kurve exakt durch den Messpunkt verläuft. Das Ergebnis erscheint als «HTs aus 3-Punkt» im Ergebnisbereich. Die 3-Punkte-Messung ist über den ⊕ 3-Punkte-Messung-Schalter aktivierbar.

6.3  SOK m ü. M. — absolute Höhenreferenz

Die zwei SOK-Felder (Mast links / Mast rechts) nehmen die absolute Schienenoberkantenhöhe in m ü. M. auf. Wichtige Regeln:

  • h₁ / h₂ bleiben immer als «über SOK»-Werte erhalten — sie werden bei SOK-Eingabe nicht umgerechnet.
  • Bei unterschiedlichen SOK-Werten links und rechts (Gefälle) interpoliert die App für jeden Diagrammpunkt linear zwischen den beiden SOK-Werten.
  • Der rechte SOK-Wert wird automatisch aus dem linken übernommen. Sobald er manuell geändert wird, bleibt er unabhängig.
  • «✕ löschen» setzt beide SOK-Felder zurück; das Diagramm kehrt zur SOK-relativen Darstellung zurück.

6.4  Diagramm & Interaktion

  • Die Y-Achse zeigt «Höhe [m ü. SOK]» resp. «Höhe [m ü. M.]» je nach SOK-Einstellung.
  • Ankerlabels an den Mastpositionen zeigen h₁/h₂ als absolute Höhe (m ü. M.) oder als «h = x.xx m».
  • Messpunkt-Callouts zeigen zweilagig: Bezeichnung (M1–M4) oben, Höhenwert unten.
  • Die orangefarbene Tiefpunkt-Karte zeigt ebenfalls m ü. M., wenn SOK gesetzt.
  • PDF-Export: Das Berechnungsprotokoll enthält alle Eingaben, Ergebnisse und das Diagramm.

6.1  Points de mesure M1 – M4

Dans la zone Points de mesure, il est possible de saisir jusqu'à quatre positions x quelconques (distances depuis le poteau gauche). L'application calcule pour chaque point:

  • Hauteur CP au-dessus DRS (et en m s.n.m. si DRS défini)
  • Hauteur FC au-dessus DRS
  • Flèche CP en ce point

Dans le diagramme, les points de mesure apparaissent comme des marqueurs colorés avec un callout. Si DRS m s.n.m. est défini, le callout affiche la hauteur absolue en m s.n.m. comme valeur principale.

6.2  Mesure 3 points

La mesure 3 points permet le calcul inverse de la force de traction CP HCP à partir d'une mesure de terrain. Saisies:

  • xC — Position de mesure (distance depuis le poteau gauche, milieu de travée recommandé)
  • yC — Hauteur CP mesurée en ce point, au choix en «m au-dessus DRS» ou en «m s.n.m.» (si DRS défini)

L'application résout numériquement pour HCP (Newton-Raphson sur la formule caténaire), de sorte que la courbe calculée passe exactement par le point de mesure. Le résultat apparaît comme «HCP de la mesure 3 pts» dans la zone résultats. La mesure 3 points s'active via le commutateur ⊕ Mesure 3 points.

6.3  DRS m s.n.m. — référence d'altitude absolue

Les deux champs DRS (Poteau gauche / Poteau droit) reçoivent la hauteur absolue du dessus du rail en m s.n.m. Règles importantes:

  • h₁ / h₂ restent toujours en valeurs «au-dessus DRS» — ils ne sont pas convertis lors de la saisie DRS.
  • En cas de valeurs DRS différentes à gauche et à droite (pente), l'application interpole linéairement pour chaque point du diagramme entre les deux valeurs DRS.
  • La valeur DRS droite est automatiquement reprise de la gauche. Dès qu'elle est modifiée manuellement, elle reste indépendante.
  • «✕ effacer» remet les deux champs DRS à zéro; le diagramme revient à l'affichage relatif DRS.

6.4  Diagramme & interaction

  • L'axe Y affiche «Hauteur [m s. DRS]» ou «Hauteur [m s.n.m.]» selon le réglage DRS.
  • Les labels d'ancrage aux positions des poteaux indiquent h₁/h₂ en hauteur absolue (m s.n.m.) ou en «h = x.xx m».
  • Les callouts des points de mesure sont sur deux niveaux: désignation (M1–M4) en haut, valeur de hauteur en bas.
  • La carte orange du point le plus bas indique également m s.n.m. si DRS est défini.
  • Export PDF: le protocole de calcul contient toutes les saisies, résultats et le diagramme.

6.1  Punti di misura M1 – M4

Nella sezione Punti di misura è possibile inserire fino a quattro posizioni x qualsiasi (distanze dal palo sinistro). L'applicazione calcola per ogni punto:

  • Altezza FP sopra PRF (e in m s.l.m. se PRF impostato)
  • Altezza FC sopra PRF
  • Freccia FP in quel punto

Nel diagramma i punti di misura appaiono come marcatori colorati con una callout. Se PRF m s.l.m. è impostato, la callout mostra l'altezza assoluta in m s.l.m. come valore principale.

6.2  Misura 3 punti

La misura 3 punti permette il calcolo inverso della forza di tiro FP HFP da una misura sul campo. Inserimenti:

  • xC — Posizione di misura (distanza dal palo sinistro, centro campata consigliato)
  • yC — Altezza FP misurata in quel punto, a scelta in «m sopra PRF» o in «m s.l.m.» (se PRF impostato)

L'applicazione risolve numericamente per HFP (Newton-Raphson sulla formula della catenaria), in modo che la curva calcolata passi esattamente per il punto di misura. Il risultato appare come «HFP da misura 3 punti» nella zona risultati. La misura 3 punti si attiva tramite il commutatore ⊕ Misura 3 punti.

6.3  PRF m s.l.m. — riferimento di altezza assoluta

I due campi PRF (Palo sinistro / Palo destro) ricevono l'altezza assoluta del piano del ferro in m s.l.m. Regole importanti:

  • h₁ / h₂ restano sempre come valori «sopra PRF» — non vengono convertiti durante l'inserimento PRF.
  • Con valori PRF diversi a sinistra e a destra (pendenza), l'applicazione interpola linearmente per ogni punto del diagramma tra i due valori PRF.
  • Il valore PRF destro viene automaticamente ripreso da quello sinistro. Non appena viene modificato manualmente, rimane indipendente.
  • «✕ elimina» azzera entrambi i campi PRF; il diagramma torna alla visualizzazione relativa al PRF.

6.4  Diagramma & interazione

  • L'asse Y mostra «Altezza [m s. PRF]» o «Altezza [m s.l.m.]» a seconda dell'impostazione PRF.
  • Le etichette di ancoraggio alle posizioni dei pali mostrano h₁/h₂ come altezza assoluta (m s.l.m.) o come «h = x.xx m».
  • Le callout dei punti di misura sono su due livelli: denominazione (M1–M4) in alto, valore di altezza in basso.
  • La scheda arancione del punto più basso mostra anch'essa m s.l.m. se PRF è impostato.
  • Esportazione PDF: il protocollo di calcolo contiene tutti gli inserimenti, risultati e il diagramma.

7  Schritt-für-Schritt Beispielberechnung (N-FL)

Aufgabe: Berechne den Fahrdraht-Durchhang und die Ts-Höhe am Feldmittelpunkt für ein ebenes N-FL-Feld.
Kettenwerk: StCu 50 mm² + Cu 107 mm², gestützt, 6 kN  |  Spannweite: L = Lm = 50 m  |  Höhen: h₁ = h₂ = 5.50 m  |  sh = 2.40 m
Zugkräfte: HTs,ub = 1 700 N  |  HFd = 8 500 N  |  Temperaturen: T₁ = +10 °C → T₂ = +40 °C

Schritt 1  —  Gesamtlast (bekannte FL-Kombination)

Aus VBA-Tabelle: MFL = 13.876 N/m (inkl. Ts + Fd + Klemmen)

Einzelwerte: qTs = 4.011 N/m  |  qFd = 9.517 N/m  |  qH = 0.196 N/m  → Summe = 13.724 N/m (ohne Klemmen). VBA-Wert MFL = 13.876 N/m (mit Klemmen) ist massgebend.

E·A = 159 GPa × 4.945 cm² = 7.862 MN  |  α = 11.5×10⁻⁶/K

Schritt 2  —  Zustandsgleichung unbelastet → belastet

Koeffizienten des kubischen Polynoms (C·H³ + K·H² − G = 0):

G = (MFL × Lm)² / 24 = (13.876 × 50)² / 24 = 20 054 N²·m
C = 1 / E·A = 1 / 7.862 MN = 1.272×10⁻⁷ m/N
K = (qTs×Lm)² / (24×H²ub) − Hub/E·A
  = (4.011×50)² / (24×1700²) − 1700/7.862×10⁶ = 3.638×10⁻⁴ m/N

Newton-Raphson (Startwert Hub = 1 700 N) → HTs,belastet = F₀ ≈ 3 860 N

Schritt 3  —  Kettenlinie bei T₁ = +10 °C

Kettenlinienparameter:

a = HTs / MFL = 3 860 / 13.876 = 278.1 m

Ebenes Feld (h₁ = h₂) → Tiefpunkt in Feldmitte: x₀ = 25.000 m

Ts-Höhe am Tiefpunkt: yTs(25 m) = h₁ − fTs

Fahrdraht-Durchhang nach VEM 7.66:

qvem = 13.876 × (1 + 8500/3860) = 13.876 × 3.202 = 44.43 N/m
fFd,1 = 50² / 8 × 44.43 / (3860 + 8500) = 312.5 × 44.43 / 12 360 = 1.123 m

Ts-Höhe SOK bei T₁: h₁ − fFd,1 + sh = 5.50 − 1.123 + 2.40 (Näherung am Tiefpunkt)

Schritt 4  —  Zustandsgleichung T₁ → T₂ (ΔT = +30 K)

VEM Handbuch / VBA Upro1 Gleichung (Newton-Raphson):

gk₀ = MFL / F₀ = 13.876 / 3860 = 3.594×10⁻³ 1/m
G  = 24/50² + gk₀² = 9.600×10⁻³ + 1.292×10⁻⁵ = 9.613×10⁻³
A  = MFL × (1 + HFd/F₀) = 13.876 × 3.202 = 44.43 N/m
α·ΔT = 11.5×10⁻⁶ × 30 = 3.450×10⁻⁴
Iteration → HTs,2 ≈ 2 880 N

Die Zugkraft nimmt bei Erwärmung ab (Ts dehnt sich aus → tiefere Spannung).

Schritt 5  —  Fahrdraht-Durchhang bei T₂ = +40 °C

fFd,2 = 50² / 8 × 44.43 / (2880 + 8500) = 312.5 × 44.43 / 11 380 = ≈ 1.220 m

Ts-Höhe SOK in Feldmitte bei T₂ ≈ 5.50 − 1.220 = ca. 4.28 m ü. SOK

Kontrolle: Minimale Fahrdrahthöhe ≥ 5.00 m ü. SOK (Fd-Höhe = Ts-Höhe − sh = 4.28 − 2.40 ≈ 5.38 m → Grenzwert eingehalten ✓)

Tipp: Das vollständige Berechnungsprotokoll mit allen Zwischenwerten und der Canvas-Darstellung kann über «PDF Berechnungsprotokoll» exportiert werden. Die Werte entsprechen dem SBB-Referenzmodell Durchhang 2026.xlsm.
